Welcome!

I've built this very simple Angular 4 application to get some practice using Angular 4, typescript and to demonstrate the use of an api.

This tiny little project relies upon an open api provided by NASA that delivers its Astronomy Photo Of the Day. You can learn more about the open api's that NASA provides HERE.

Given that I am hosting this very simple app on Github Pages, I have opted not to include my own NASA api key here and have instead used the default key provided by NASA. Using this key comes with some restrictions, specifically 30 requests per IP address per hour and 50 requests per IP address per day.

If you are not getting an example of the NASA APOD on the second link to the right, these restrictions may be causing the issue.
